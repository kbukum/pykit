# SBOM — Software Bill of Materials

pykit generates a CycloneDX-format SBOM for every release. The SBOM is:

- Attached to each [GitHub release](https://github.com/kbukum/pykit/releases) as `sbom.cyclonedx.json`
- Generated by [anchore/sbom-action](https://github.com/anchore/sbom-action)

To inspect: `cat sbom.cyclonedx.json | python -m json.tool | head -50`
